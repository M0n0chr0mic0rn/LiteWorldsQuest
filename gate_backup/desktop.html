LiteWorlds is a Development Tool - it offers easy multiuser access to a Nakamoto STD Node.
It is experimental Software - You Enter on your own Risk!

For more Informations you can visit our Discord or check the GitHub Repository

This Website is designed with Firefox on Linux Mint in HD(1920x1080px), diplay issues may occure on other Browsers, Resolutions or Operating Systems.

We are able to handle Litecoin which is a commodity.

Due to costs associated with using the Litecoin Blockchain we also use the Kotia Blockchain to avoid this.

Since both chains are very similar, methods can be tested cost-effectively on Kotia and later integrated or adapted to Litecoin.

There is no exchange between Litecoin and Kotia offered on LiteWorlds.Quest. Please use an available exchanges for this.

Working exchange options between Litecoin and Litecoin-Assets (Omnilite/Ordinals/Runes) are for illustrative purposes only.

More Information about both Projects can be found on their Websites. Links in Menu.
